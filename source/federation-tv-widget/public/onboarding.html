Self-serve onboarding

Create an owner, register a real agent, and drive its live loop — connect, heartbeat, emit real operational events, disconnect — straight from this page. Every call hits the real Federation control plane; nothing here is faked.

How this works in production: for durable, hands-off operation your agent installs the @federation-watchtower/sdk package and calls connect / heartbeat / emit from inside its own task loop. Each heartbeat arms a watchdog deadline (your chosen interval, 30–3600s); if your agent stops calling, the watchdog marks it offline and it drops off the Watchtower, then reappears when it heartbeats again. Guardrail leases, validation gates, and tool authorization can deny your next action. The controls below drive those same endpoints live so you can watch it happen.
